Como funciona?

O módulo transparente para Magento permite que o cliente digite os dados de pagamento no seu próprio site. Antes mesmo de finalizar o pedido, esses dados são enviados para o PagSeguro UOL de forma segura, gerando um hash único da transação. Este é o hash transacionado, fazendo com que a transação ainda seja segura em qualquer ambiente, pois seu servidor não armazenará, transmitirá ou processará nenhuma informação sensível do cliente.

Opções de instalação

Instalar com modgit

                $ cd pasta/do/magento
                $ modgit init
                $ modgit add pagseguro git@github.com:r-martins/PagSeguro-Magento-Transparente.git

Instalar com modman

                $ cd pasta/do/magento
                $ modman init
                $ modman clone git@github.com:r-martins/PagSeguro-Magento-Transparente.git

Ou manualmente

  • Faça download da última versão aqui
  • Descompacte na raíz da instalação da loja mesclando com as pastas existentes
  • Limpe o cache
  • Se não for utilizar o modelo de aplicação, é necessário solicitar a autorização para uso da API transparenteneste link.
  • Certifique-se que não exista nenhum outro módulo PagSeguro instalado. Se existir, remova-o ou desative-o via XML.

Configurações no Painel do PagSeguro

  • Integrações->Pagamentos pela API: desativado
  • Integrações->Notificação de Transações: Ativado. Coloque o url http://www.SUALOJA.com.br/index.php/pseguro/notification
  • Se você não for utilizar o modelo de aplicação, lembre-se de autorizar sua loja a usar a api transparente. Em caso de demora ou dúvidas sobre a liberação ligue na Central de Atendimento do PagSeguro: (11)5627-3440 (Segunda a sábado das 8h às 20h30, exceto feriados).

Detecção de problemas

A maior parte dos problemas pode ser identificada ativando o modo debug no painel do magento e vendo o arquivo de log gerado na sua pasta/do/magento/var/log/pagseguro.log.

Funcionalidades

  • PagSeguro com suporte a cartão de crédito, boleto e TEF(Débito em conta) permitindo digitar os dados sem a necessidade de ser redirecionado para o site do PagSeguro.
  • Trata retorno automático, atualizando status do pedido.
  • Cálculo automático de parcelas de acordo com o cartão selecionado e as regras definidas no seu painel do PagSeguro.
  • Mapeamento de atributos como CPF, Data de Nascimento, rua, bairro, baseado em outros já existentes
  • Modo de debug com logs detalhados.
  • Modo de testes usando Sandbox **
  • Token criptografado nas configurações. Maior segurança pra você.
  • Não salva dados do cartão. Apenas os últimos 4 dígitos.
  • Pagamentos com Boleto ou TEF. *

* Disponível apenas nas versões PRO ou APP

** Não disponível na versão APP

Versão Gratuita

Disponível gratuitamente no GitHub ou Magento Connect. Este módulo deve estar presente também se optar por contratar o PRO ou APP.

Para utilizar, você precisa que sua conta esteja liberada junto ao PagSeguro para utilizar a API transparente.

O pedido de liberação pode levar alguns dias, e pode ser solicitado nesse link sem nenhum custo.

PRO

Deve ser instalado em conjunto com a versão gratuita e é enviado pra você via e-mail logo após a contratação.

Adiciona opções de Boleto e TEF à sua loja.

  • Valor: R$9,90/mês
  • Forma de Pagamento: Cartão de crédito, utilizando sistema de assinatura do próprio PagSeguro.
  • Pode ser cancelado a qualquer momento, sem nenhum custo adicional, diretamente no seu painel do PagSeguro.

Contratar PRO

APP (ou Modelo de Aplicação) - 14 ou 30 dias

Deve ser instalado em conjunto com a versão gratuita e é enviado pra você via e-mail logo após a autorização. Neste modelo, você autoriza a "aplicação", que no caso é o Módulo PagSeguro transparente, a realizar transações com a sua conta.

Somente no modelo APP não é necessário ter a autorização do PagSeguro para utilizar a API transparente.

Você deve instalar este módulo junto com a versão gratuita acima.

Ao optar pelo modelo de aplicação, você receberá seus pagamentos podendo optar por receber em 14 dias, com um desconto na tarifa normal. Veja:

  • Valor: 0,5% cobrado sobre todas as transações aprovadas, porém, você ganha um desconto de 0,5% na tarifa do PagSeguro. *
  • * Ao invés de pagar 4,99%, você passará a pagar 4,79%, ou seja, usa o módulo gratuitamente, e ainda tem um desconto. :)
  • Forma de Pagamento: descontado automaticamente nas vendas aprovadas, sem necessidade de cartão.
  • Pode ser cancelado a qualquer momento, sem nenhum custo adicional, diretamente no seu painel do PagSeguro ou através deste link.
